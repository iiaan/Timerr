# Timerr

## Qué es

Timerr es una aplicación simple para **bloquear el acceso a la red durante un tiempo determinado**, ayudándote a concentrarte en tareas sin distracciones. Cuenta con un **temporizador integrado** que controla automáticamente cuándo levantar el bloqueo.

## Funcionalidades

- Bloquea la red de tu computadora temporalmente.  
- Contador regresivo para el tiempo de bloqueo.  
- Interfaz mínima y directa: abrir, cronometro, activar, esperar.  

## Cómo usar

1. Ejecuta `Timerr.exe` (o desde Visual Studio si quieres compilar desde código fuente) > ⚠️ SIEMPRE USAR COMO ADMINISTRADOR.  
2. Activa el bloqueo.  
3. La red se desbloqueará cuando el usuario decida.
